# A Simple Game

## Description

A simple 2D game that was made as a practice of working with coordinate systems, keyboard event handling and DOM manipulation.

## Technologies

- HTML
- CSS
- JavaScript

## How to Run

You can use this project in one of the following ways:

### Option 1: Use the Live Version

Simply open [this link](https://simple-game-rouge.vercel.app/) in your browser to access the application online.

### Option 2: Run Locally

1. Clone the repository:

```bash
git clone https://github.com/KreimerR/simple-game.git
```

2. Navigate to the project folder:

```bash
cd simple-game
```

3. Open `index.html` in your browser to start the game.

## How to Play

Use the arrow keys to move the player (red square) across the grid and collect all coins (yellow squares). The game ends when all coins are collected.

## Features

- Grid-based coordinate system
- Randomized coin generation
- Keyboard movement handling
- Score tracking
- Win condition detection
